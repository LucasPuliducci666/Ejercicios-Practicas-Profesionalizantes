Definiciones:

HTML: Es un lenguaje de programacion utilizado para crear paginas web estaticas. Es quien permite estructurar el contenido de un sitio mediante distintas etiquetas unicas

Navegador: Es una aplicación que permite el acceso al internet. Interpreta el código HTML para mostrarlo de forma visual al usuario.

Estilo: Serie de propiedades que se le puede brindar a una etiqueta para modificar sus caracteristicas. Sirve para darle vida a la pagina y a sus elementos, o mismo diferenciar secciones de otras con paletas de color.
